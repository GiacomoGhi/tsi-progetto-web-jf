## IDE Suggested Extensions

In Visual Studio Code click the Extensions button in the Activity Bar and install the following extensions:

- Babel ES6/ES7
- Better Comments
- Debugger for Chrome
- DotENV
- ESLint
- Prettier Code Formatter
